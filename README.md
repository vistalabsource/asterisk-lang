# asterisk-lang
現在開発中のスクリプト言語。

# 概要
前回のBascよりもっと機能を追加しました。
コーディングは**ほぼほぼCodexに任せた**のでバグがあるかもしれないです。

# 基本機能
変数
`$変数名: "Hello";`  
モジュール機能 
`load "モジュール名";`  
if/elseif/else
```
if (条件式) {
    処理...
} elseif (条件式) {
    処理...
} else {
    処理...
}
```
while文
```
while (条件式) {
    処理...
}
```
関数定義
```
function 関数名: $引数1, $引数2... {
    処理...
}
```
リスト
` [値1, 値2, 値3...];`
for文
```
for ($変数: 繰り返したい内容) {
    処理...
}
```
辞書
` {キー: 値...} `
タプル
` (値1, 値2, 値3...) `

# データ型
文字列型（シングルクォート非対応） ` "Hello" `  
整数型/浮動小数点型 ` 123, 3.14 `  
ブール型 ` True, False `
